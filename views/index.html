I've got another api for you!

Use this api to make an image search that return the url, alt-text and page-url in the JSON format.
Simple make a http request to "https://image-api-fcc.glitch.me/search/[your search]".
This supports pagination using the query "?offset=[number]".

Example: https://image-api-fcc.glitch.me/search/carrotcake?offset=2

That's not all! You can see the most recent searches but making a http request to https://image-api-fcc.glitch.me/recent.
You can even adjust how many results you see with the "?queries=[number]" query.

Example: https://image-api-fcc.glitch.me/recent?queries=2
